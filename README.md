# Tafor

[![Build Status](https://travis-ci.org/up1and/tafor.svg?branch=master)](https://travis-ci.org/up1and/tafor)

本软件用于民航气象部门智能发布预报报文、趋势报文、重要气象情报，监控预报报文，以声音或电话的方式返回告警。

## 功能
- TAF 报文编辑
- TREND 报文编辑
- SIGMET 报文编辑
- 监控与告警
    - 自动 METAR、TAF 报文入库查询
    - TAF 报文迟发告警
    - 报文发布提醒
